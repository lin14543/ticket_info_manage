# ticket_info_manage
航班信息数据的Web管理展示平台
# 复制本项目
```
git clone https://github.com/lin14543/ticket_info_manage
```
# 搭建虚拟环境并激活
```shell
virtualenv py2
source py2/bin/activate
```
# 安装所需要的包
进入到本项目的根目录下
```
pip install -r requirements.txt
```
# 在settings里修改自己的配置
＃启动项目
python manage.py runserver 0.0.0.0:8000
